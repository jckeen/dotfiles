# Scoping (issue #214): every plugin here is INSTALLED on every machine, but
# only the [global] section is ENABLED in ~/.claude/settings.json

# (posthog ~150 skills, soundcheck ~60, vercel ~30) that drown the curated
# repo skills when enabled globally — they're enabled per-project via each
# project's committed `.claude/settings.json` instead. The section markers
# are plain comments, so setup.sh / sync-plugins.sh install both sections
# unchanged; PluginDriftCheck.hook.ts parses them to warn (never block) when
# live enablement drifts from this layout.
